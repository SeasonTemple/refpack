# Example SkillHub Catalog

This directory contains a minimal SkillHub catalog shape. The catalog is intentionally file-backed and read-only for v1.

`catalog.json` references a `.tgz` artifact path relative to the configured artifact root:

```text
browser-agent/1.0.0.tgz
```

For a real local run, generate that artifact from `examples/basic-skill-pack` and update `integrity` and `sizeBytes` to match the generated bytes:

```bash
node ../../dist/cli.js skillhub pack ../basic-skill-pack --version 1.0.0 --out ./artifacts
node ../../dist/cli.js skillhub validate --catalog ./catalog.json --artifact-root ./artifacts
```

Tests generate deterministic artifacts at runtime rather than committing binary archives.
